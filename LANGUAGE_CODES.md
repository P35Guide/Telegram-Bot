# Language Codes Quick Reference / Коди мов - Швидкий довідник

## Supported Language Codes / Підтримувані коди мов

```
✓ en      - English (default / за замовчуванням)
✓ ua, uk  - Українська / Ukrainian
✓ de      - Deutsch / German
✓ fr      - Français / French
✓ es      - Español / Spanish
✓ it      - Italiano / Italian
✓ pl      - Polski / Polish
✓ pt      - Português / Portuguese
✓ ja      - 日本語 / Japanese
✓ zh      - 中文 / Chinese
```

## How to change language / Як змінити мову

### In the bot / У боті:
1. Open Settings / Відкрийте Налаштування: **⚙️ Settings** / **⚙️ Налаштування**
2. Select Language / Виберіть Мову: **🌐 Language** / **🌐 Мова**
3. Choose from the list / Виберіть зі списку

### Automatic detection / Автоматичне визначення:
- The bot automatically detects your Telegram language
- Бот автоматично визначає мову з ваших налаштувань Telegram

## Special mappings / Спеціальні мапінги

```
ru → en  (Russian → English / Російська → Англійська)
be → en  (Belarusian → English / Білоруська → Англійська)
ua → uk  (Alternative Ukrainian code / Альтернативний код української)
```

## Default language / Мова за замовчуванням

**English (en)** - If your language is not supported, English will be used automatically.

**Англійська (en)** - Якщо ваша мова не підтримується, автоматично використовується англійська.
